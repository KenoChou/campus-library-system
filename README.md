# Campus Library Management System

Modern Apple-style Campus Library Management System Prototype.

> A modern library management platform designed for schools, colleges and universities.

## 🚀 Live Demo

Coming Soon

## ✨ Features

### User Authentication & Permissions

* User Login
* Role-Based Access Control (RBAC)
* Administrator Management
* Student & Teacher Access

### Reader Management

* Reader Information Management
* Search & Filter
* Borrowing History
* Reader Status Tracking

### Book Management

* Book Catalog Management
* ISBN Management
* Category Management
* Inventory Tracking

### Borrow & Return

* Book Borrowing
* Book Returning
* Borrowing Records
* Renewal Management

### Overdue Reminder

* Due Date Monitoring
* Overdue Tracking
* Notification Center

### Analytics Dashboard

* Borrowing Statistics
* Popular Books Ranking
* Reader Activity Analysis
* Library Operation Overview
